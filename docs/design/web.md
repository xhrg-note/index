## web相关的一些规范




#### httpcode
http的返回码一般会有200，404，500。
如果是业务异常比如说用户登录失败，那么需要返回200。如果是系统异常就返回500，一个很好的判断依据是。如果这个错误需要程序员介入，那就是500，不需要介入则是500



#### http返回数据

http返回数据，一般喜欢用code,msg,data三个属性。code有三种可能性，1. 成功2000，2.异常需要程序员介入。3.异常不需要程序员介入。返回的msg表示错误弹出信息。data是成功的数据。数据格式一般使用下划线，而不是驼峰。比如person_id，而不是personId
